## FPMS 개요
FPMS는 Foot Pressure Measure System의 약자입니다. 족부의 압력 분포를 세밀하게 측정하여 이를 기반으로 보행 습관을 분석하여 사용자에게 건강 정보와 기타 제안을 제시하는 시스템입니다.
